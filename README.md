# 💰 FinanzRechner

Sammlung deutschsprachiger Finanzrechner für GitHub Pages – kein Backend, kein API-Key.

🔗 **Live:** `https://<dein-username>.github.io/finanz-rechner/`

---

## Rechner-Übersicht

| Seite | Themen |
|-------|--------|
| [Zinseszins](pages/zinseszins.html) | Kapitalwachstum, monatliche Sparrate, Meilenstein-Tabelle, Szenarien |
| [Kredit & Annuität](pages/kredit.html) | Rate / Laufzeit / Kreditbetrag berechnen, vollständiger Tilgungsplan |
| [Sparrechner](pages/sparen.html) | Endwert, Spardauer oder Sparrate ermitteln, Inflationsbereinigung |
| [Altersvorsorge](pages/rente.html) | Rentenlücke, Kapitalbedarf, notwendige Zusatzsparrate |
| [Einkommensteuer DE](pages/steuer.html) | § 32a EStG 2024: Grundtarif, Splittingtarif, Soli, Kirchensteuer, SV |
| [Inflationsrechner](pages/inflation.html) | Kaufkraftverlust, zukünftiger Preis, Halbwertszeit, hist. Daten DE |
| [ROI / NPV / IRR](pages/investition.html) | Return on Investment, Kapitalwert, Interner Zinsfuß, Cashflow-Analyse |

---

## Hinweis

Alle Berechnungen dienen der Orientierung und ersetzen keine Steuer- oder Finanzberatung. Steuerformeln entsprechen dem deutschen EStG Stand 2024.

---

## Deployment

```
Settings → Pages → Source: main / root
```

Kein npm, kein Build, kein Backend. Reines HTML/CSS/JS.
